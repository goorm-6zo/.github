# 서비스 소개
👤 얼굴 인식 기반 출입 관리 시스템
비접촉식 얼굴 인식 기술을 활용해 행사 입장부터 퇴장까지의 과정을 자동화한 시스템입니다.
기존의 수기 확인이나 QR 코드 방식 대신, 얼굴만으로 신속하고 정확한 출입 인증이 가능하며, 실시간 방문객 관리도 지원합니다.
